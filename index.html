This generator operates by:
1. Convert the string to a binary string using the alphabet:
  ['a','b','c','d','e','f','g','h','i','j','k','l','m','n','o','p','q','r','s','t','u','v','w','x', 'y','z','0','1','_','.',',','\'']
2. Determine the ideal splitting of the letters based on the length of the key
  a. Maximizes the least common multiple of any number of numbers that add up to key length or less
3. Split the binary string into groups
4. Stack the groups of binary strings on top of each other
5. XOR the bits at each position
6. Convert the generated key back to text using the alphabet

In the graphs if there's fewer digits than expected there is a 0 in front (0 = 00 = 000)
